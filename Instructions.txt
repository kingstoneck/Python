Instructions for Ubuntu Server
  -clean install
  -set up 3 partitions as follows (200GiB HDD):
    /sqldb/
      -150 GiB, for SQL database
    /trader/
      -25 GiB, for general trading items.  Create a transfer folder here
    /
      -remaining space for system files
      
Update
  sudo apt-get update
Install updates
  sudo apt-get upgrade
  
Add new users:
  sudo adduser 'user'
  transfer
    -user for transfering files to and from the system
  trader
    -user for trading, will need SSH access
  sql
    -user to monitor and manager SQL remotely
    
  #note, no premissions set at this time
  
Grant user sudo access:
  sudo usermod -aG sudo 'user'

Install MySQL  #instructions from https://www.digitalocean.com/community/tutorials/how-to-install-mysql-on-ubuntu-20-04
  sudo apt install mysql-server
Secure the install  
  sudo mysql_secure_installation
    -validate password strength
    -set strength to STRONG (option 2)
    -set root password
    -remove root access

Move the DB to the new folder #instrustions from https://www.digitalocean.com/community/tutorials/how-to-move-a-mysql-data-directory-to-a-new-location-on-ubuntu-16-04
  sudo mysql -u root -p
    #requires sudo as root cannot access the SQL server.  Must log in as someone other than root
